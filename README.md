# Smart-Incubator-ESP32
 Automated Smart Incubator Simulation (ESP32)

An embedded systems project designed to monitor ambient temperatures and automatically actuate a servo-controlled safety ventilation hatch if thresholds are crossed. Built completely in simulation using the ESP32 microcontroller.

## How it Works
* **Microcontroller:** ESP32 running C++
* **Input:** DHT22 Sensor tracking ambient real-time temperature.
* **Output 1:** I2C LCD Screen displaying real-time data status.
* **Output 2:** Servo Motor acting as an automated ventilation physical hatch.
